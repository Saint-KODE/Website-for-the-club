Upcoming Events

AWS SUMIT

The AWS Summit is a free, global event series hosted by Amazon Web Services (AWS), designed to educate and connect cloud computing enthusiasts, developers, and businesses. It features keynote speeches, technical sessions, hands-on labs, and networking opportunities, helping attendees learn about the latest AWS innovations, cloud best practices, and real-world use cases. AWS Summits are held in major cities worldwide, catering to both beginners and advanced users looking to expand their cloud expertise..

DevConf Johannesburg 2025

Hackathons

Coding Club Web Dev Hackathon

The hackathon challenge is to create a dynamic and beginner-friendly website for the Coding Club, showcasing the club’s mission, events, projects, and resources. The site should be engaging, with sections like a welcoming homepage, an about page, event listings, and a showcase for member projects. Social media links should be included, and participants are encouraged to incorporate creative UI/UX design, responsiveness, and interactive features to enhance the user experience.
